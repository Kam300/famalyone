# Отладка импорта данных

## Проблема с вашим JSON

В вашем JSON файле есть запись:

```json
{
  "id": 55,
  "firstName": "Жена",
  "lastName": "Шамиль",
  "fatherId": 12,
  "motherId": 11
}
```

**Проблема:** ID 12 и 11 отсутствуют в JSON файле! Это означает, что родители "Жены Шамиль" не будут найдены при импорте.

## Что происходит при импорте:

### Шаг 1: Создание маппинга ID
```
Старый ID → Новый ID
37 → 1
38 → 2
39 → 3
50 → 4
41 → 5
52 → 6
43 → 7
44 → 8
55 → 9
47 → 10
```

### Шаг 2: Восстановление связей
- **Альбина (ID 37)**: Отец 38 ✅, Мать 44 ✅ → Связи восстановлены
- **Марат (ID 38)**: Отец 41 ✅, Мать 43 ✅ → Связи восстановлены
- **Шамиль (ID 39)**: Отец 41 ✅, Мать 43 ✅ → Связи восстановлены
- **Лилия (ID 44)**: Отец 50 ✅, Мать 52 ✅ → Связи восстановлены
- **Жена Шамиль (ID 55)**: Отец 12 ❌, Мать 11 ❌ → Связи НЕ восстановлены
- **Ребёнок Шамиля (ID 47)**: Отец 39 ✅, Мать 55 ✅ → Связи восстановлены

## Что добавлено в код:

### 1. Логирование в Logcat
Теперь вы можете видеть в Android Studio (Logcat):
```
Import: Starting import of 10 members
Import: Member: Альбина Гарипова, ID: 37, Father: 38, Mother: 44
Import: Mapped: old ID 37 -> new ID 1 (Альбина Гарипова)
Import: ID Mapping complete: 10 entries
Import: Father ID 12 not found for Жена Шамиль
Import: Mother ID 11 not found for Жена Шамиль
Import: Links updated: 5, failed: 1
```

### 2. Информация в диалоге
После импорта показывается:
```
Добавлено: 10
Связей восстановлено: 5
⚠️ Связей не восстановлено: 1
(родители отсутствуют в файле)
```

## Решение:

### Вариант 1: Исправить JSON
Добавьте в JSON родителей с ID 11 и 12, если они существуют.

### Вариант 2: Убрать несуществующие связи
Измените запись "Жены Шамиль":
```json
{
  "id": 55,
  "firstName": "Жена",
  "lastName": "Шамиль",
  "fatherId": null,
  "motherId": null
}
```

## Как проверить:

1. Запустите приложение
2. Откройте Logcat в Android Studio
3. Фильтр: `Import`
4. Импортируйте JSON
5. Смотрите логи - там будет видно, какие связи не восстановлены

## Ожидаемый результат:

Для вашего JSON:
- ✅ Добавлено: 10 человек
- ✅ Связей восстановлено: 5 (Альбина, Марат, Шамиль, Лилия, Ребёнок)
- ⚠️ Связей не восстановлено: 1 (Жена Шамиль - родители ID 11, 12 отсутствуют)
