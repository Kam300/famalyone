# 🚀 Быстрое руководство - Семейное Древо v1.3.0.3

## ✨ Что нового

### 1. Выбор шаблона отображения древа

**Два стиля на выбор:**

#### 📱 Современный стиль
```
┌────────┐  ━━  ┌────────┐
│ [Фото] │      │ [Фото] │
│ Иванов │      │Иванова │
│  Иван  │      │ Мария  │
└────────┘      └────────┘
```
- Полная информация
- Прямоугольные карточки
- Детальный просмотр

#### 🌳 Классический стиль
```
   ⭕  ━━  ⭕
  Иван   Мария
 Иванов Иванова
```
- Круглые фото
- Компактный вид
- Традиционное древо

### 2. Улучшенное управление

- ✅ Видимый горизонтальный скроллбар
- ✅ Плавная прокрутка
- ✅ Вертикальная прокрутка для больших древ

## 🎯 Как использовать

### Переключить шаблон:
1. Откройте **"Древо семьи"**
2. Нажмите **📋** в правом верхнем углу
3. Выберите стиль
4. Готово! ✨

### Прокрутить древо:
- **Горизонтально:** свайп влево/вправо
- **Вертикально:** свайп вверх/вниз
- Скроллбары видны всегда!

## 📊 Семейные пары

Древо автоматически показывает связи:

```
[Отец] ━━━ [Мать]
        ▼
   [Дети под парой]
```

### Как создать пару:
1. Добавьте **отца**
2. Добавьте **мать**
3. Добавьте **ребенка** и укажите обоих родителей
4. Они автоматически станут парой! 👨‍👩‍👧

## 🎨 Когда использовать

### Современный стиль:
- Длинные имена и фамилии
- Детальный просмотр
- Печать с данными
- Повседневное использование

### Классический стиль:
- Презентация семейной истории
- Большие семьи (>10 человек)
- Декоративный постер
- Традиционный вид

## ⚡ Быстрый старт

1. **Установите приложение**
2. **Добавьте первого члена семьи** (например, себя)
3. **Добавьте родителей** (укажите роли)
4. **Свяжите** - при добавлении себя укажите родителей
5. **Откройте древо** - увидите пару родителей!
6. **Попробуйте шаблоны** - нажмите 📋

## ❓ FAQ

**Q: Как создать пару?**  
A: Добавьте ребенка и укажите обоих родителей при создании.

**Q: Как поменять шаблон?**  
A: Откройте древо → нажмите 📋 → выберите стиль.

**Q: Сохранится ли выбор?**  
A: Да, автоматически сохраняется и восстанавливается.

**Q: Можно ли прокручивать?**  
A: Да! Горизонтально и вертикально, скроллбары видны.

**Q: Сколько поколений можно добавить?**  
A: Неограниченно! От прапрадедушек до правнуков.

**Q: Нужен ли интернет?**  
A: Нет! Все работает локально, интернет только для обновлений.
