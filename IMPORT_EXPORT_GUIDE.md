# Импорт/Экспорт данных семьи

## Реализовано

### Импорт данных из JSON
- Кнопка "Импорт данных" в настройках
- Чтение JSON файла с членами семьи
- Автоматическое добавление в базу данных
- Без фотографий (только текстовые данные)

### Формат JSON для импорта

```json
[
  {
    "firstName": "Иван",
    "lastName": "Иванов",
    "patronymic": "Иванович",
    "gender": "MALE",
    "birthDate": "01.01.1980",
    "role": "FATHER",
    "phoneNumber": "+79001234567",
    "fatherId": null,
    "motherId": null,
    "weddingDate": "15.06.2005"
  }
]
```

### Поля:
- `firstName` (обязательно) - Имя
- `lastName` (обязательно) - Фамилия  
- `patronymic` - Отчество
- `gender` (обязательно) - MALE или FEMALE
- `birthDate` (обязательно) - Дата рождения (ДД.ММ.ГГГГ)
- `role` (обязательно) - Роль (FATHER, MOTHER, SON, DAUGHTER, и т.д.)
- `phoneNumber` - Телефон
- `fatherId` - ID отца (null если неизвестен)
- `motherId` - ID матери (null если неизвестна)
- `weddingDate` - Дата свадьбы

### Использование:
1. Откройте Настройки
2. Нажмите "Импорт данных"
3. Выберите JSON файл
4. Данные автоматически добавятся в приложение
